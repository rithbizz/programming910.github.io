Part 1: The if Statement

What is a Conditional Statement?

A conditional statement tells Python: â€œOnly do this if something is trueâ€

Real-life examples of conditions:

  • If the score is above 50 â†’ you pass

# With conditions, code runs ONLY when True

    print("You are an adult.")   # Only runs if age >= 18

ğŸ’¡ Key Idea

Conditional statements let your program respond differently to different situations.

Syntax of the if Statement

The structure of an if statement:

    # code to run when condition is True
    # this block is INDENTED (4 spaces or 1 tab)

Rules to remember:

  • The condition is followed by a colon :
  • The code inside must be indented (press Tab or 4 spaces)
  • Indented code is called the body or block

    print("It is hot today!")
    print("Stay hydrated.")

print("This always runs.")   # Not indented = outside the if

âš ï¸ Indentation is REQUIRED

Python uses indentation to know what belongs inside the if. Missing or wrong indentation causes an error!

Indentation: The Most Important Rule

# âœ… CORRECT â€” indented with 4 spaces

if score >= 50:
    print("You passed!")   # 4 spaces before print

# âŒ WRONG â€” no indentation â†’ IndentationError

print("You passed!")       # ERROR! Must be indented

# âŒ WRONG â€” inconsistent indentation

  print("You passed!")     # 2 spaces
    print("Great job!")    # 4 spaces â†’ ERROR!

ğŸ¯ Golden Rule

Everything inside an if block must have the SAME indentation level. Use 4 spaces consistently (or Tab â€” but not both).

How Python Evaluates the Condition

Python checks if the condition is True or False:

# Condition is True â†’ block RUNS

    print("x is greater than 5")    # âœ… This runs

# Condition is False â†’ block is SKIPPED

Part 6: Inline if (Ternary Expression)

Shorthand ifâ€¦else

For simple one-line decisions, Python has a compact syntax:

Syntax:

ğŸ’¡ When to use it

Use the ternary form when the choice is simple and fits on one readable line. For complex logic, always use a full ifâ€¦else for clarity.

Ternary Examples

# Example 2: Absolute value

print(f"Absolute value: {absolute}")  # 15

# Example 3: Max of two numbers

maximum = a if a > b else b
print(f"Maximum: {maximum}")         # 25

# Example 4: Greeting
hour = 14
greeting = "Good morning" if hour < 12 else "Good afternoon"
print(greeting)                      # Good afternoon

Summary: if, elif, else

What youâ€™ve learned today:

Concept Purpose Syntax
if Run code only when True if condition:
else Run code when if is False else:
elif Check additional conditions elif condition:
Nested if Decisions within decisions Indent deeper
Combined conditions and, or, not if a and b:
Ternary One-line shorthand x if cond else y

ğŸ¯ Key Takeaways

  • Python reads conditions top to bottom and stops at the first True
  • Indentation is mandatory â€” 4 spaces per level
  • Use elif (not multiple if) when choices are mutually exclusive
  • Use parentheses to make complex conditions clear
  • Test all your boundary values (the exact cutoff numbers)!
